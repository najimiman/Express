définition de mongodb:
MongoDB est un système de gestion de base de données document-orienté, open source et NoSQL. 
Il a été conçu pour stocker et gérer des données non structurées et semi-structurées, ainsi que des données structurées. 
MongoDB utilise des documents JSON (JavaScript Object Notation) pour représenter les données stockées, 
ce qui permet une grande flexibilité dans la structure des données.
